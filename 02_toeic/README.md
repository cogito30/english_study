## 목표설정
- 취업을 위한 기준 점수(750+)
- 개인목표(800+)

## 학습법
- LC
```
1) 문제풀기
2) 지문확인후 오답분석(근거 찾기)
3) 틀린 문제/정확하게 듣지 못한 문제 다시 듣고 받아쓰기
4) 모르는 단어 암기
5) 3-4 과정 반복
```
- RC
```
1) 지문읽기
2) 문제풀고 오답분석(근거 찾기)
3) 해석이 안되는 문장 체크후 단어 암기와 함께 반복 연습
```
