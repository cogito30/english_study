# TOEIC Sentence 100 (11-20)

#### 11
- 첫째로
```
First of all
```
- 첫쨰로, 어린이들의 장난감은 안전하고 깨끗해야 합니다.
```
First of all, childeren's toys should be safe and clean
```

#### 12
- 이러한 이유로
```
For these reasons
```
- 이러한 이유들로, 저는 라디오를 듣는 것보다 TV를 보는 것을 선호합니다.
```
For these reasons, I prefer watching television rather than listening to the radio
```

#### 13
- ~에 매우 만족하다
```
be very happy with
```
- 저는 지난 학기 성적에 매우 만족했습니다.
```
I was very happy with my grades for the last semester
```

#### 14
- ~하는 것을 즐기다
```
I enjoy ~ing
```
- 저는 여가 시간이 있을 떄 영화를 보는 것을 즐깁니다.
```
I enjoy watching movies when i have free time
```

#### 15
- ~하도록 유혹을 받다
```
be tempted to 부정사
```
- 저는 집에서 일할 때, 가끔 TV를 보거나, 컴퓨터를 하거나, 아니면 무언가를 먹도록 유혹을 받습니다.
```
When i work at home, i am sometimes tempted to watch TV, paly the computer, or get something to eat
```

#### 16
- ~이기 때문에 ~을 좋아하다
```
i like ~ because
```
- 운동하는 것은 저를 건강하게 유지해주기 때문에 저는 운동을 좋아합니다.
```
i like working out because it keeps me healthy
```

#### 17
- B보다 A를 선호하다
```
I prefer A to B
```
- 저는 기차를 타는 것보다 버스를 타는 것을 선호합니다.
```
i prefer taking a bus to taking a train
```

#### 18
- 거의 ~않다
```
I rarely
```
- 저는 전자 제품 상점에 거의 가지 않습니다.
```
I rarely go to electronic stores.
```

#### 19
- ~하고 싶다
```
I would like to 부정사
```
- 저는 휴가에 태국을 가고 싶습니다.
```
I would like to go to Thailand for my vacation
```

#### 20
- 차라리 ~하겠다
```
I would rather
```
- 저는 약속을 정하거나 회의를 잡을 때 차라리 전화를 하겠습니다.
```
I would rather call a person on the phone when making appointments or scheduling meetings
```
