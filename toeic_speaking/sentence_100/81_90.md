# TOEIC Sentence 100 (81-90)

#### 81
- ~하기 위한 좋은 방법
```
a good way to부정사
```
- 이것은 상품을 홍보하기 위한 좋은 방법일 것입니다.
```
This could be a good way to advertise our products
```

#### 82
- ~보다 나은
```
better than
```
- 저는 대가족인 것보다 소가족인 것이 낫다고 생각합니다.
```
I think having a small family is better than having a large family
```

#### 83
- ~하는 것이 어협다고 느끼다
```
find it difficult to부정사
```
- 저는 정장을 입어야 할 때, 불편하고 어떤 것에도 집중하기 어렵다고 느낍니다.
```
When I have to wear a suit, I feel uncomfortable and find it difficult to concentrate on anything
```

#### 84
- ~일지라도
```
even though
```
- 저는 급여가 적을지라도 제가 즐기는 직업을 갖는 것이 낫다고 생각합니다.
```
I think it is better to have a job that I enjoy even though it pays less
```

#### 85
- 예를 들면
```
For example
```
- 예를 들면, 저는 어렸을 때 항상 10시까지 잠자리에 들었습니다.
```
For example, when I was a child I always went to bed by 10 o'clock
```

#### 86
- ~하게 되다
```
get to부정사
```
- 외국에서 공부할 때, 제가 배우는 언어를 연습하게 됩니다.
```
When I study abroad, I get to practice the language I learn
```

#### 87
- 나는 ~에 ehddmlgkek
```
I agree that
```
- 여성들이 남성들과 동일하게 급여를 받아야 한다는 것에 동의합니다.
```
I agree that women should be paid equally to men
```

#### 88
- B보다 A를 선호하다
```
I prefer to A rather than B
```
- 저는 집에서 일하는 것보다 사무실에서 일하는 것을 선호합니다.
```
I prefer to work at an office rather than work at home
```

#### 89
- ~외에도
```
In addition to
```
- 영화를 보는 것 외에도, 영화관에는 제가 즐기는 다른 것들이 있습니다.
```
In addition to watching a movie, there are other things I enjoy at a movie theater
```

#### 90
- A뿐만 아니라 B도
```
not only A, but also B
```
- 월요일뿐만 아니라 화요일에도 워크숍이 있을 것입니다.
```
There will be a workshop not only on Monday, but also on Tuesday
```
