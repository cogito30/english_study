# TOEIC Sentence 100 (61-70)

#### 61
- 정각에 도착하다
```
be on time
```
- 자동차 문제로 정각에 도착하지 못할 것입니다.
```
I will not be on time becaused of car trouble
```

#### 62
- ~할 충분한 시간
```
enough time to부정사
```
- 발표를 위한 준비를 끝낼 충분한 시간이 없었습니다.
```
There isn't enough time to finish the perparations for the presentation
```

#### 63
- ~하는 데에 어려움을 겪다
```
have difficulty -ing
```
- 저는 귀하의 메시지를 들었고, 비행기를 예약하는 데에 어려움을 겪고 있다는 것을 알았습니다.
```
I just listened to your message and understand that you are having some difficulty booking flights
```

#### 64
- ~을 추천하다
```
I recommend that
```
- 저는 약속을 다시 잡는 것을 추천합니다.
```
I recommend that we reschedule the appointment
```

#### 65
- ~에 대해 진심으로 사과하다
```
I sincerely apologize for
```
- 실수에 대해 진심으로 사과드립니다.
```
I sincerely apologize for the mistake
```

#### 66
- ~이라면 부담 갖지 말고 제게 연락 주세요
```
Please feel free to contact me if you have any questions
```
- 질문이 있으시면 부담 갖지 말고 제게 연락 주세요.
```
Please feel free to contact me if you have any questions
```

#### 67
- ~을 제안하다
```
I suggest that
```
- 저는 귀하꼐서 그 프로젝트를 도울 보조를 고용할 것을 제안합니다.
```
I suggest that you hire an assistant to help you with that project
```

#### 68
- ~인지 알아보겠다
```
I will see if
```
- 오늘 밤에 그들이 귀하의 업무르 도울 수 있는지 알아보겠습니다.
```
I will see if they are free to help you out with your presentation
```

#### 69
- 기꺼이 ~할 것이다.
```
I would be happy to
```
- 저는 발표에 대해 기꺼이 귀하를 도울 것입니다.
```
I would be happy to assist you with your presentation
```

#### 70
- ~에 대해 듣게 되어 유감이다.
```
I'm sorry to hear about
```
- 귀하가 구매하신 장비와 관련된 문제에 대해 듣게 되어 유감입니다.
```
I'm sorry to hear about your trouble with the equipment you purchased
```
