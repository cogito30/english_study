# TOEIC Sentence 100 (01-10)

#### 01
전반적으로 ~ 인 것 같다
```
Basically, it seems like ~
```
- 전반적으로, 사람들은 잔디 위에서 쉬고 있는 것 같습니다.
```
Basically, it seems like the people are relaxing on the grass
```

#### 02
- ~처럼 보이다
```
be seen as
```
- 깨긋하고 유행하는 옷을 입은 사람은 부유하고 똑똑해 보입니다.
```
A person wearing clean, fashionalbe clothing is seen as rich and smart
```

#### 03
- 평균적으로
```
On average
```
- 평균적으로, 저는 일 년에 두 번 학교를 위한 새 신발을 사요
```
On average, I buy new shoes for school twice a year
```

#### 04
- 전반적으로, ~인 것처럼 보이다
```
Overall, it appears to be
```
-  전반적으로, 도시에서의 바쁜 하루인 것처럼 봉빈디ㅏ.
```
Overall, it appears to be a busy day in the city
```

#### 05
- 요약하자면
```
To summarize
```
- 요약하자면, 사람들이 따뜻한 코트를 입고 있기 때문에 추운 날인 것 같습니다.
```
To summarize, it seems like a cold day because the people are wearing warm coats
```

#### 06
- ~하는 것이 더 쉽다
```
it is easier to 부정사
```
- 큰 차보다 작은 차를 주차할 공간을 찾는 것이 더 쉽습니다.
```
it is easier to find a parking space for a small vehicle than a large one
```

#### 07
- 언제든지
```
at any time
```
- 저는 케이블 TV에 여러 개의 영화 채널이 있어서, 언제든지 다양한 영화를 볼 수 있습니다.
```
I have several movie channels on my cable TV system, so i can watch a variety of movies at any time
```

#### 08
- ~할 것 같은
```
be likey to 부정사
```
- 저는 단체로 박물관을 관람할 것 같습니다.
```
i am likely to tour a museum with a tour guide
```

#### 09
- ~하는 것은 ~하다
```
it is ~ to 부정사
```
- 편지를 쓰는 것보다 이메일을 보내는 것이 더 쉽습니다.
```
it is easier to send an e-mail than write a letter
```

#### 10
- 혼자서
```
by oneself
```
- 저는 보통 혼자서 박물관에 갑니다.
```
i usually go to museums by myself
```
