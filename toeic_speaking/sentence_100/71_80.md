# TOEIC Sentence 100 (71-80)

#### 71
- 다른 질문이 있다면
```
If you have any other questions
```
- 다른 질문이 있으시면, 555-1217로 부담 갖지 말고 제게 전화하십시오.
```
If you have any other questions, please feel free to call me back at 555-1217
```

#### 72
- 기꺼이 ~하다
```
I am willing to부정사
```
- 귀하가 발표 준비하는 것을 제가 기꺼이 도와 드리겠습니다.
```
I am willing to help you prepare the presentation
```

#### 73
- 원한다면, ~하겠다
```
If you'd like, I will
```
- 원하시면, 예산에 대해 귀하를 도와드릴 수 있도록 제 비서를 귀하의 사무실로 보내겠습니다.
```
If you'd like, I will send my assistant up to your office to help you with the budget
```

#### 74
- 그동안에
```
In the meantime
```
- 그동안에, 새로운 직원들은 일하기 위해 화의실을 이용할 수 있습니다.
```
In the meantime, the new staff can use the conference rooms to do their work
```

#### 75
- A가 B하는 것을 돕다
```
help A with B
```
- 아이들은 그들의 부모가 집안일 하는 것을 도와야 합니다.
```
Children should help their parents with the household chores
```

#### 76
- 만약 선호한다면, ~할 수 있다.
```
If you prefer, I can
```
- 만약 선호하시면, 제가 바로 은행으로 서류들ㅇ르 보낼 수 있습니다.
```
If you prefer, I can send the documents directly to the bank
```

#### 77
- ~인 것 같다
```
it sounds like
```
- 안타깝게도, 저희가 귀하를 위해 설치한 기계에 결함이 있는 것 같습니다.
```
Unfortunately, it sounds like the machine we installed for you may be defective
```

#### 78
- ~에 문제가 있다
```
have a problem with
```
- 저는 귀하께서 저희 상점에서 최근에 구입한 에어컨에 문제가 있다는 것을 알았습니다.
```
I understand that you are having a problem with the air conditioner that you recently purchased from our store
```

#### 79
- ~을 보상하다
```
make up for
```
- 바라건대 이것이 오류가 야기한 불편을 보상할 것입니다.
```
Hopefully, this will make up for any inconvenience the error may have caused
```

#### 80
- ~에 대한 제 사과를 받아 주십시오.
```
Please accept my apology for
```
- 불편에 대한 제 사과를 받아 주십시오.
```
Please accept my apology for the inconvenience
```
