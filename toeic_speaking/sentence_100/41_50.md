# TOEIC Sentence 100 (41-50)

#### 41
- ~하는 데에 누구의 시간을 쓰다
```
take one's time -ing
```
- 저는 도서관에 책을 반납하는 것에 대해 걱정할 필요가 없고 책을 읽는 데에 시간을 쓸 수 있습니다.
```
I don't have to worry about returning the book to the library and can take my time reading it
```

#### 42
- ~하기로 되어 있다
```
be supposed to
```
- 행사 소개와 설명을 위해 귀하는 9시 30분에 대회의실로 가기로 되어 있습니다.
```
You are supposed to go to the main board room a 9:30 for an introduction and explaination of activities
```

#### 43
- 죄송하지만
```
I'm afraid
```
- 죄송하지만, 이번 해에 댄스 공연은 없을 것입니다.
```
I'm afraid there won't be a dance performance this year
```

#### 44
- ~에게 (to부정사)할 것을 부탁하다
```
ask ~ to부정사
```
- 저는 제 비서에게 오늘 아침에 사무용품을 가져다 달라고 부탁했어요
```
I asked my secretary to bring the office supply this morning
```

#### 45
- ~을 저희에게 알려주십시오
```
Please let us know
```
- 4월 18일 금요일까지 귀하가 무엇을 가져올 수 있는지 제게 알려주십시오
```
Please let us know what you can bring by Friday, April 18
```

#### 46
- ~에/~에서 열리다
```
take place at/on/in
```
- 행사는 5월 9일에 열립니다.
```
The ceremony will take place on May 9
```

#### 47
- 첫 번째 ~은 -이다
```
The first ~ is -
```
- 첫 번째 수업은 목요일마다 오후 6시에 있는 초급 스페인어입니다.
```
The first class is "Beginners Spanish" on Thursdays at 6:00 P.M.
```

#### 48
- ~이 있다
```
There is/are
```
- 화요일에 이용 가능한 두 가지 종류의 차가 있습니다.
```
There are two types of cars available on Tuesday
```

#### 49
- ~에/~에서 열리다
```
be held at/on/in
```
- 발표는 회으실에서 열릴 것입니다.
```
The presentation will be held in the board room
```

#### 50
- ~하는데 (금액)이 들다
```
It costs (금액) to부정사
```
- 도시에 사는 것은 돈이 많이 듭니다.
```
It costs a lot of money to live in the city
```
