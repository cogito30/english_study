# TOEIC Sentence 100 (91-100)

#### 91
- 그것은 ~때문이다
```
That's because
```
- 그것은 학생들이 교육비에 대해 걱정할 필요가 없기 때문입니다.
```
That's because students don't have to worry about their education expenses
```

#### 92
- ~할 여유가 되다
```
can afford to부정사
```
- 대부분의 사람들은 의료비를 위해 적은 돈을 낼 여유가 됩니다.
```
Most people can afford to pay a small amount for medical care
```

#### 93
- ~에 대해 확신하다
```
be sure of 
```
- 젊은 사람들은 그들의 결정에 확신하지 못하므로, 다른 사람의 조언을 필요로 합니다.
```
Young people are not sure of their decisions, so they need the advice of others
```

#### 94
- ~하도록
```
so that
```
- 다른 사람들을 방해하지 않도록 사람들은 식당에서 휴대전화를 사용하지 않아야 합니다.
```
People should not use mobile phones in restaurants so that they don't disturb others
```

#### 95
- 이유는 ~이다
```
The reason is that
```
- 이유는 어린이와 노인들은 요금을 낼 여유가 되지 않기 때문입니다.
```
The reason is that children and older people cannnot afford to pay the fee
```

#### 96
- 구체적으로 말하자면
```
To be specific
```
- 구체적으로 말하자면, 학생들은 무엇을 입을지 걱정하는데 시간을 덜 쓸 것입니다.
```
To be specific, student will spend less time worrying about what to wear
```

#### 97
- 결과적으로
```
as a result
```
- 저는 운동을 많이 해서 결과적으로 건강합니다.
``` 
I exercise a lot and am in good shape as a result
```

#### 98
- ~에 더하여
```
as well as
```
- 저는 여가 시간이 있을 때 축구에 더하여 야구와 농구를 하는 것을 좋아해요
```
I like playing baseball, and basketball, as well as soccer when I have some free time
```

#### 99
- ~하는 사람들
```
those who
```
- 연회에 참석하고 싶은 사람들은 예약을 해야만 합니다.
```
Those who wish to attend the banquet must make reservations
```

#### 100
- ~하도록 강요하다
```
be forced to부정사
```
- 젊은이들은 팀 스포츠에 참여하도록 강요되어서는 안 됩니다.
```
Young people should not be forced to participate in team sports
```
