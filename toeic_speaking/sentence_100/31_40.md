# TOEIC Sentence 100 (31-40)

#### 31
- 그러므로
```
Therefore
```
- 그러므로, 사람들은 정기적으로 운동을 해야합니다.
```
Therefore, people should exercise regularly
```

#### 32
- ~하는 데에 시간을 낭비하다
```
waste time -ing
```
- 저는 콘서트 표를 사려고 줄 서 있는 데에 시간을 낭비하고 싶지 않습니다
```
I don't like wasting my time standing in a line to buy concert tickets
```

#### 33
- ~에게 어울려 보이다
```
look good on
```
- 만약 정장이 너무 크거나 작으면, 사람들에게 어울려 보이지 않습니다
```
If a suit is too big or small, it doesn't look good on people
```

#### 34
- ~과 비교하여
```
compared to
```
- 어렸을 때와 비교하여 저는 지금 TV를 덜 봅니다
```
I watch less TV now compared to when I was younger
```

#### 35
- ~해야 한다
```
have to
```
- 저는 출장을 자주 가야 합니다.
```
I have to travel a lot for business
```

#### 36
- A와 B 둘다
```
both A and B
```
- 자신의 차를 갖는 것은 편리하고 시간이 절약됩니다.
```
Owning my own vehicle is both convenient and time-saving
```

#### 37
- ~하는 것이 필수적이다
```
it is necessary to 부정사
```
- 학생들의 질문에 명확하게 답하는 것은 필수적입니다.
```
It is necessary to answer student's questions clearly
```

#### 38
- ~할 필요가 있다
```
need to 부정사
```
- 부모들은 반드시 자녀들이 훌륭한 예절을 갖추도록 할 필요가 있습니다.
```
Parents need to make sure their children have good manners
```

#### 39
- ~하는 데에 (얼마의) 시간을 쓰다
```
spend (시간)-ing
```
- 저는 평균적으로 운동하는 데에 일주일에 세시간 정도를 씁니다.
```
I spend about three hours per week exercising on average
```

#### 40
- ~과 같은
```
such as
```
- 저는 결혼식이나 면접과 같은 경우에만 정장을 입습니다.
```
I only wear a suit on occasions such as weddings or job interviews
```
