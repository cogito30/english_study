# TOEIC Sentence 100 (51-60)

#### 51
- 당신은 잘못 알고 있다
```
you're mistaken
```
- 그 정보에 대해 귀하께서 잘못 알고 계신 것 같습니다.
```
I think you're mistaken about that information
```

#### 52
- ~에 예정된
```
scheduled for
```
- 오후에 예정된 세 개의 행사가 있습니다.
```
There are three events scheduled for the afternoon
```

#### 53
- ~할 것이 요구되다
```
be asked to부정사
```
- 모든 가족들은 사이드 요리, 샐러드, 또는 후식을 가져올 것이 요구됩니다.
```
All families are asked to bring a side dish, salad, or dessert
```

#### 54
- ~에 계획된
```
planned for
```
- 여행의 마지막 날에 계획된 세 가지의 행사가 있습니다.
```
There are three activities planned for the final day of the tour
```

#### 55
- ~에 함께 포함되다
```
be included with
```
- 2일용 출입증은 입장료에 점심이 함께 포함되어 있어서 비쌉니다.
```
The two-day pass is expensive because a picnic lunch is included with the cost of admission
```

#### 56
- 더 이상 ~이 아닌
```
no longer
```
- 충분한 자유 시간이 없기 때문에 저는 더 이상 해외로 여행을 많이 가지 않습니다.
```
I no longer travel abroad much because I don't have enough free time
```

#### 57
- ~에 대해 사과하다
```
I am sorry for
```
- 다시 한 번, 불편에 대해 사과드립니다.
```
Again, I am sorry for any inconvenience
```

#### 58
- 가능한 빨리
```
as soon as possible
```
- 가능한 빨리 귀하으 ㅣ댁에 저희의 기술자 중 한 명을 보내 드리겠습니다.
```
I will send one of our technicians over to your home as soon as possible
```

#### 59
- ~할 수 있다
```
be able to부정사
```
- 그러면, 귀하는 오늘 저녁에 저녁 서비스를 위해 분을 열 수 있을 것입니다.
```
That way, you will be able to open for dinner service this evening
```

#### 60
- ~을 알다
```
I understand that
```
- 귀하의 구매에 문제가 있다는 것을 알았습니다.
```
I understand that you have a problem with your purchase
```
